---
title: Updating
---

You can update `dehub` either with the [GitHub CLI](https://cli.github.com) or by
downloading the binaries manually.

## Update Using `gh`

To update `dehub` simply run:

```sh
gh extension upgrade dlvhdr/gh-dehub
```

You should see an output similar to:

```sh
[dehub]: upgraded from v4.16.2 to v4.17.0
✓ Successfully checked extension upgrades
```

Run `gh extension --help` for info on more commands.

## Update Manually

To manually update `dehub`, you can download the binaries from the latest release on GitHub under:
https://github.com/dlvhdr/gh-dehub/releases

Search for the binary that corresponds to your operating system and CPU
architecture.
