## `↑/k` - Move Up

Press <kbd>↑</kbd> or <kbd>k</kbd> to move to the previous work item in the current section.

## `↓/j` - Move Down

Press <kbd>↓</kbd> or <kbd>j</kbd> to move to the next work item in the current section.

## `[` - Previous Section

Press <kbd>[</kbd> to move to the previous section in the current view.

## `]` - Next Section

Press <kbd>]</kbd> to move to the next section in the current view.

## `h/home` - First Item

Press <kbd>h</kbd> or <kbd>Home</kbd> to move to the first work item in the current section.

## `g/end` - Last Item

Press <kbd>g</kbd> or <kbd>End</kbd> to move to the last work item in the current section.
