## `p` - Cycle Preview Pane

Press <kbd>p</kbd> to cycle the preview pane between vertical split, horizontal split, and hidden.

## `ctrl+down` - Preview Page Down

Press <kbd>Ctrl</kbd>+<kbd>↓</kbd> to shift the view for the preview pane down one step. The first line in
the updated view is always the line after the last line of the previous view.

## `ctrl+up` - Preview Page Up

Press <kbd>Ctrl</kbd>+<kbd>↑</kbd> to shift the view for the preview pane up one step. The last line in the
updated view is always the line before the first line of the previous view.

## `ctrl+left` - Previous Preview Tab

Press <kbd>Ctrl</kbd>+<kbd>←</kbd> to move to the previous tab in the preview sidebar, if one exists.

## `ctrl+right` - Next Preview Tab

Press <kbd>Ctrl</kbd>+<kbd>→</kbd> to move to the next tab in the preview sidebar, if one exists.
