## `p` - Cycle Preview Pane

Press <kbd>p</kbd> to cycle the preview pane between vertical split, horizontal split, and hidden.

## `ctrl+right` - Focus Preview Pane

Press <kbd>Ctrl</kbd>+<kbd>→</kbd> to focus the preview pane when it is open.

## `ctrl+left` - Focus Main Pane

Press <kbd>Ctrl</kbd>+<kbd>←</kbd> to return focus to the main pane.

## `up`/`down` - Scroll Preview

When the preview pane is focused, press <kbd>↑</kbd>/<kbd>↓</kbd> or <kbd>k</kbd>/<kbd>j</kbd> to scroll the preview.

## `h`/`g` - Preview Top/Bottom

When the preview pane is focused, press <kbd>h</kbd> to jump to the top or <kbd>g</kbd> to jump to the bottom.

## `left`/`right` - Previous/Next Preview Tab

When the preview pane is focused, press <kbd>←</kbd> or <kbd>→</kbd> to move between preview tabs, if tabs are available.
