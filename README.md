# gh-dash

A terminal UI for GitHub, built from the combined work of [dlvhdr/gh-dash](https://github.com/dlvhdr/gh-dash) and [pierrecomputer/pierre diffshub](https://github.com/pierrecomputer/pierre).

## Docs

See [gh-dash.dev/getting-started](https://gh-dash.dev/getting-started) for usage and configuration docs.

## Install From Source

Make sure you have `gh`, Go, and Bun installed, then run:

```bash
make local-install
```

This builds the bundled diff viewer, builds `gh-dash`, and installs the current checkout as the `dash` GitHub CLI extension.
